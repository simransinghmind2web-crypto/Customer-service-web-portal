Here's the fully functional In-Service Support (ISS) Web System, built directly from the process flow in your PPTX. Here's what's included:
Three main sections:

Report Snag — The ISS submission form captures trainer ID, reporter details, snag description, severity level (Low / Medium / High / Critical), preferred resolution date, and contact info. Submitting auto-generates a ticket (e.g. ISS-0004) and simulates routing to ISS@aerotraincorp.com.
All Complaints — A live register of all submitted ISS forms with search and status filters. Clicking any row opens a detailed modal with the full process timeline and action buttons to advance the ticket through the workflow.
Process Flow — A visual step-by-step reference of all 9 stages (Snag → Reported → Emailed → Acknowledged → Fixed → Documented → Verified → Logged → Closed), with roles clearly marked (CFSATE / Aerotrain Corp / Both).

Key features:

Ticket progression: Open → Acknowledged → In Progress → Verified → Closed
Live dashboard stats (Open, In Progress, Resolved, Total)
Persistent storage via localStorage (survives page refresh)
3 demo tickets pre-loaded for demonstration
Fully responsive for mobile/tablet use
